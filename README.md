# Student-Performance-Analysis

## 📌 Project Overview
This project analyzes student performance using Jupiter notebook. The goal is to understand how different factors such as gender, lunch type, and test preparation affect student scores.

---

## 🎯 Objectives
- Analyze student scores (math, reading, writing)
- Compare performance across different categories
- Visualize data using charts
- Generate meaningful insights

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset
The dataset contains 1000 student records with the following features:
- Gender
- Race/Ethnicity
- Parental Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

---

## 📊 Analysis Performed
- Data cleaning and validation
- Feature engineering (average score)
- Group-wise analysis using `groupby()`
- Correlation analysis
- Data visualization

---

## 📈 Visualizations
- Heatmap (Correlation)
- Bar Chart (Gender vs Scores)
- Histograms (Score Distribution)
- Pie Chart (Average Score Contribution)

---

## 🔍 Key Insights
- Average student performance is moderate (~67.77)
- Reading and writing scores are higher than math
- Female students perform better than male students
- Students with standard lunch perform better
- Test preparation improves performance significantly
- Strong correlation exists between subjects
